# E-Commerce Website 🌐

A modern, responsive product showcase website built using HTML and CSS.

Technologies Used
- HTML5
- CSS3
- Flexbox for layout
- Media Queries for responsive design

## 📱 Responsive Design
This site is fully responsive:
- Optimized for large screens 💻
- Fully functional on mobile devices 📱
